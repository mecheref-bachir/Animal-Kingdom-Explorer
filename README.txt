Features

    Explore Animal: Users can browse through a detailed list of animals, viewing specific information like habitat, diet, and unique characteristics.
    Explore Species:  Users can browse through a detailed list of species, viewing specific information like name, and description characteristics.

Tech Stack

    Kotlin: Primary programming language for the app.
    Android SDK: User interface design and interaction.
    Room Database: For data persistence.
    LiveData and ViewModel: To observe data changes and manage UI-related data responsibly.
    Navigation Component: For managing in-app navigation.
